3，Prometheus 计算量大的时候，会很卡，可以试试 https://github.com/Comcast/trickster 这个缓存加速器。

        docker run --name trickster -d  tricksterio/trickster \
          && docker cp trickster:/usr/local/bin/trickster . \
          && docker rm -f trickster
